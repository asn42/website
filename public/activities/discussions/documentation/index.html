Documentation

Il arrive souvent que l'on découvre des choses intéressantes lors de discussions. Malheureusement, il arrive aussi relativement souvent qu'on les oublie.

Nous essaierons de remettre au propre les notes des discussions dans cette section.

Logiciel Libre

Nous avons préparé un guide pour vous aider à trouver un bon premier bug à résoudre et faire une contribution à un logiciel Libre. Vous vous demandez peut-être si vous pouvez être payé pour vos contributions à des logiciels Libres, ou des ressources et des informations sur le logiciel libre.

Sécurité

Nous pouvons essayer de vous donner des pistes si vous voulez vous lancer dans la sécurité, ou si vous voulez travailler dans la sécurité informatique. Vous pouvez aussi aller voir les sources d'information que nous avons rassemblé sur le sujet.
